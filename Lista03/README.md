## Exercício 1

### RC4
Para compilar, utilize o comando:
```
gcc rc4.c -o rc4.out
```
Para executar, utilize
```
./rc4.out
```
O resutado estará no arquivo **output_rc4.txt**.


### Salsa20
Para compilar, utilize o comando:
```
gcc salsa20.c -o salsa20.out
```
Para executar, utilize
```
./salsa20.out
```
O resutado estará no arquivo **output_salsa.txt**.
